# 영어 카드뉴스 생성기 (insta-cardnews)

매일의 영어 패턴(표현 + 예문 5개)을 **인스타그램 카드뉴스**(1080×1080 정사각)로 만들어 주는 생성기입니다.
브라우저에서 열기만 하면 되고, 별도 설치가 필요 없습니다.

## 구성

```
insta-cardnews/
├── index.html      # 카드 생성기 (더블클릭으로 실행)
└── patterns.js     # 문장 데이터 (표현 60개 × 예문 5개, en/ko)
```

## 사용법

1. `index.html`을 브라우저(Chrome 권장)로 엽니다.
   - 폰트(Google Fonts)와 저장 기능(html2canvas)은 인터넷 연결이 필요합니다.
2. 열면 **오늘 날짜에 해당하는 패턴**이 자동으로 카드 7장(인트로 + 예문 5 + 마무리)으로 만들어집니다.
3. 상단 드롭다운에서 다른 패턴을 골라 만들 수도 있습니다.
4. **[이 카드 저장]** 또는 **[전체 PNG 저장]** 으로 PNG를 내려받아 인스타그램에 여러 장(캐러셀)으로 업로드합니다.

## 디자인

- 파스텔 소프트 테마, 예문 카드마다 색상 순환(핑크·옐로·민트·스카이·라벤더)
- 영어(Nunito) + 한글(Noto Sans KR) 병기, 문장 길이에 따라 글자 크기 자동 조절
- 계정 표기: `@Eng.bite_`

## 데이터

`patterns.js`의 `window.PATTERNS` 배열을 편집하면 카드 내용이 바뀝니다.
각 항목 형식:

```js
{ "pattern": "표현 ~", "meaning": "뜻", "examples": [ { "en": "영어 예문", "ko": "한글 뜻" }, ... 5개 ] }
```

이 데이터는 데일리 영어 발송 프로젝트(`daily-english-kakao`)의 `patterns.json`과 동일한 내용입니다.
그 프로젝트에서 패턴을 갱신하면, `patterns.json` → `patterns.js`로 다시 생성해 동기화하면 됩니다.
